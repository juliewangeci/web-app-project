# SugarGrain Bakery

## Project Overview
A fully responsive, multi-page web application built using HTML 5, CSS3, JavaScript (ES6+) and Bootstrap 5. This project fulfills the requirements outlined in the " Web application assignment"

## Theme
SugarGrain Bakery is a realistic shpwcase website for artisanal neighborhood bakery specializing in fresh-baked breads, and fine pastries.

## Planned pages (7 minimum)
1. Home ('index.html')- Welcome banner to the bakery, daily specials, and brand intro.
2. About Us ('about.html')- Story of our bakery and how it came to be.
3. Menu ('menu.html')- Organized, responsive breakdown of the goods that the bakery has to offer
4. Gallery ('gallery.html')- high-quality photos of custom orders.
5. Contact & Custom orders ('contact.html')- Form for custom cake requests.
6. Testimonials ('reviews.html')- customer feedback and submission form
7. FAQ's ('faq.html')- deliver, pre-order, and allergen details.

## Planned interactive features (JavaScript)
**Feature 1:** Dynamic custom order form validation(On contact page)
**Feature 2:** Filterable gallery/menu to sort items by category (eg: gluten-free, vegan, pastries).
